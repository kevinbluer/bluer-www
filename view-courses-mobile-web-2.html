Building Successful Web and Mobile Applications November 2015

In this 10 week-course we explore what it takes to be successful web applications (using Meteor.js), from prototyping the early concept through to the actual build and ultimate deployment. Throughout the course students will prototype and build their own individual applications, all the while contributing to a “real-world” project that we would build collaboratively and launch...

  1. Defining a Succesful Web Application
  2. Prototyping and Idea Validation
  3. Client vs Server (and Language Introduction)
  4. Introducing AngularJS
  5. Introducing Node.js and Meteor
  6. Meteor Core Concepts and Package Management
  7. Real World Meteor Example
  8. Meteor and Mobile (and Project Development)
  9. Deployment and Hosting
  10. Application Scaling & Growth
